Selectors

Element

Selects elements based on the element name (ex: p, h1, body)

ID

Uses the id attribute of an HTML element to select a specific element (ex: #para1)

Class

Selects elements with a specific class attribute (ex: .center, p.center)

Grouping

Style multiple elements with the same style definitions, separted with commas (ex: h1, h2, p).
